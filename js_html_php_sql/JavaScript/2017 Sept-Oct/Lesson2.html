При вводе двух чисел в верхние окошки можно нажать кнопку pressToCalculate и увидеть значение сложной функции, которую можно посмотреть в скрипте.
При вводе значения от 0 до 100 в нижнее поле ползунок ставится в соответствующее положение. Если хочется измененить значение ползунка вручную и увидеть результат в цифрах, нужно нажать кнопку "Range value to textfield", и тогда значение ползунка отобразится в нижнем окне ;)
